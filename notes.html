Cascade order of attribute selectors:

  1. element
  2. class
  3. attribute
  4. id
If you assign all four of these selectors to an element, it will ultimately go with the id selector. As it is at the bottom of the cascade.


Three different ways to apply CSS to a file.

  • external uses a external style sheet in project folder. (ex. style.css)
  • internal uses style element inbetween open and close brackets of style element. This will only exist within the one webpage it is written in.
  • inline style is applied through the style attribute inside the opening tag of an html element.
If all three of these methods are used, the inline style will take priority as it is at the bottom of the cascade.


***The important key word will take priority over all the rules relative to that element.

Example:
color: red;
color: green !important;
The cascade will display green because it has been applied with the important keyword.
